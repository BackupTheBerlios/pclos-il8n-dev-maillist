command-not-found:
-----------------

Tool that informs PCLinuxOS user about missing program availability.
This utility was inspired by Ubuntu's command-not-found package but it's
implemented in a different way.

Example:

[root at localhost /]# vit
Program "vit" is not installed.
It's part of the following package:
---------------------------------------------------------------------
Package:                 vit-1.1-1pclos2007.noarch
Description:             Video Installation Tool for ATI & Nvidia
Install by typing:       su -c 'apt-get install vit'
---------------------------------------------------------------------
