Hello all,

I've cleaned up all vit translations using msgmerge:

msgmerge -U --backup=simple cs.po vit.pot

This way I've added new messages from vit.pot to all .po files and
messages not used anymore in vit have been commented out.

There are two new messages in .po files: VIT desktop icon name and
comment. So, if you want to translate vit .desktop file texts to your
language, just grab your .po file and fill in the msgstrs. I'm going to
release a new vit version soon.
