> Testing the translation is quite easy - just compile it by typing:
>
> msgfmt vit.po -o vit.mo
>
> Then copy the file vit.mo to directory
> /usr/share/locale/sv_SE/LC_MESSAGES/ and launch vit to check it - it
> should talk to you in your language.
> It the case of vit, you need version vit-1.1 from testing, former
> versions are not internationalized.
> This procedure is universal and should work for all internationalized
> applications.
