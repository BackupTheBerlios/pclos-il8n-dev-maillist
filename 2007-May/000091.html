ok since the explanation i gave you on the main forum arn't enough, here 
comes some more directions


po files :
-> here are the reference files : 
http://cvs.berlios.de/cgi-bin/viewcvs.cgi/pclos-il8n/CVSROOT/po/fr/

these are the french translation of files needed by applications to 
translate onscreen messages
they are composed of pairs of strings; 1 in english, a second in  
french. grab the one that you want to translate and replace the french 
string with a russian translation of the french one

-> here are the translated po for your locale : 
http://cvs.berlios.de/cgi-bin/viewcvs.cgi/pclos-il8n/CVSROOT/po/**/ 
(replace xx by your language code)

this is where translated and partly translated to your language files 
will be stored for the community to have access to them

how to edit those po ? install kbabel trough synaptic

use it as your main application for translating po files, its interface 
is prety straight forward and will be prety easy to nderstand

desktop files :

-> here are the reference files : 
http://cvs.berlios.de/cgi-bin/viewcvs.cgi/pclos-il8n/CVSROOT/desktop-files/

-> here is an answer to the question HOWTO? : 
http://didouph.free.fr/wiki?TranslatingIcons

documentation / other : just ask and you'll get answer, reference files 
are here : 
http://cvs.berlios.de/cgi-bin/viewcvs.cgi/pclos-il8n/CVSROOT/other/

check the "original" subfolder to fall on the original pclinuxos 
"embeded" minimal documentation wich needs to be translated in as many 
language as possible.

you to start slow then hit big ...
