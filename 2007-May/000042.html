IMPORTANT NOTICE FOR ALL

i would like all of you to check if your files/patches are based on the 
last version in cvs before submitting changes and to make sure your 
changes are consistants with the latest version of the cvs and don't 
overwrite a yet applyed change ... i just pulled down a lot of work done 
on the pl translation due to such mistake

a good way to proceed is when translating .desktop / .directory files is 
to simply send me a text file formated as follow :
--------------------------------
/pathtofile/filename.desktop

Name[yourlocale]=whatevername
Comment[yourlocale]=whatevercomment
--------------------------------
this makes things simpler for me to update the desktop definitions ad 
keep a clean cvs three

check the latest nl-be translation for a good example on how to do those 
files !
